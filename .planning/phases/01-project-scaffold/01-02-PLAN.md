1. Scaffold the Vite vanilla JS project from the project root:

```bash

npm create vite@latest frontend -- --template vanilla
```

This creates `frontend/` with the standard Vite vanilla template (index.html, src/main.js, src/counter.js, src/style.css, package.json, etc.).

2. Install dependencies:

```bash
cd /Users/Sheppardjm/Repos/dontCave/frontend
npm install
```

3. Create `frontend/vite.config.js` with the proxy configuration:

```javascript
import { defineConfig } from 'vite'

export default defineConfig({
  server: {
    proxy: {
      '/api': {
        target: 'http://localhost:8000',
        changeOrigin: true,
        rewrite: (path) => path.replace(/^\/api/, ''),
      },
    },
  },
})
```

This proxies all `/api/*` requests from the frontend to the backend at `localhost:8000`, stripping the `/api` prefix. So `fetch('/api/health')` in the browser hits `localhost:8000/health` on the backend.

4. Replace the contents of `frontend/src/main.js` with:

```javascript
import './style.css'

document.querySelector('#app').innerHTML = `
  <h1>dontCave</h1>
  <p id="status">Checking backend...</p>
`

fetch('/api/health')
  .then(r => r.json())
  .then(data => {
    document.querySelector('#status').textContent =
      `Backend: ${data.status}`
  })
  .catch(err => {
    document.querySelector('#status').textContent =
      `Backend: unreachable (${err.message})`
  })
```
